### Before Installation

Please start backend server

### Installation

Install the dependencies

```sh
$ npm install
```

```sh
$ npx tsc
```
### Create Symlink

```sh
$ npm link
```

### Commands

List Cars (list)
```sh
$ car-cli list
```

Find Cars (sort)
```sh
$ car-cli sort [options]
```

Add Car (add)
```sh
$ client-cli create
```

Update Car (update)
```sh
$ client-cli update [ID]
```

Delete Car (delete)
```sh
$ client-cli delete [ID]
```
